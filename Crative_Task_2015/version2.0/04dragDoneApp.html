Drag Done iOS App

Focus on five recurring tasks

  • create a daily, weekly, or monthly folder*
  • create five tasks in a folder
  • take a picture and / or name the task
  • optionally set a reminder
  • drag the task and it's done

*) The tasks in the folder reset based on the chosen frequency. The achievments get saved in the stats.
